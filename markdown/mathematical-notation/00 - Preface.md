# Preface

Mathematics is the language of science and engineering. Like all languages, it has its own special vocabulary. When confronted with an unfamiliar word from its lexicon, we can search for that term in our textbooks or on the web using tools such as [Wolfram Mathworld](http://mathworld.wolfram.com/). But what do we do when confounded by some strange arrow or squiggle? Just how would one type $\mapsto$ or $\ni$ into a search engine?

This guide collects the mathematical symbols and notation commonly used in science and engineering. It is based on Edward Scheinerman's *Mathematical Notation: A Guide for Engineers and Scientists* (2011), substantially reworked for a software engineer learning mathematics: each chapter now opens with historical context, weaves in programmatic demonstrations (SymPy, Z3, NumPy, JAX), and bridges notation to Python equivalents throughout.

## How to use this guide

You can use this guide in two ways:

- **As a reference.** If you encounter an unfamiliar symbol, find it by topic — the chapters are organized by mathematical domain (Letters, Collections, Logic, Numbers, Geometry, Functions, Linear Algebra, Calculus, Probability, Approximation). The Chart and Notation Index chapters at the end provide symbol-by-symbol lookup.

- **As a companion.** Read a chapter front-to-back when you're entering a new area of mathematics. The historical hooks give you context for *why* the notation exists, and the code demos let you interact with each concept immediately.

Every chapter connects notation to code. If you can read Python, you can read the math — the notation is just a more compact way to express the same operations.

## What this guide is not

Do not expect to learn new mathematics from this guide; that's not the goal. It provides enough information to *remind* you about notation you may have previously encountered, or to give you enough context to search for a deeper reference.

This guide does not address discipline-specific notation. For example, chemists use square brackets to denote concentration, but this is a convention from chemistry, not mathematics, and so it is beyond scope. We also do not cover notation for physical constants such as $\hbar$ (Planck's constant divided by $2\pi$).

## Further reading

For the history of how mathematical notation evolved, Florian Cajori's *A History of Mathematical Notations* (1928) is the definitive reference — two volumes covering everything from Babylonian numerals to modern operator notation. Jeff Miller's [Earliest Uses of Various Mathematical Symbols](http://jeff560.tripod.com/mathsym.html) is a lighter online companion. For the comprehensive catalogue of $\mathrm{LaTeX}$ symbols, see Scott Pakin's [Comprehensive LaTeX Symbol List](http://www.ctan.org/tex-archive/info/symbols/comprehensive/). The full bibliography is in the Bibliography chapter.
